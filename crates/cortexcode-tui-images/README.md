# cortexcode-tui-images

Terminal image rendering for the cortex TUI: capability detection, Kitty
graphics protocol and iTerm2 inline image protocol encoding, and
lightweight PNG/JPEG/GIF/WebP dimension sniffing.

Part of the [cortexcode](https://github.com/kolisachint/cortexcode) Rust workspace.

Ported from `@kolisachint/hoocode-tui`'s `terminal-image.ts`.
